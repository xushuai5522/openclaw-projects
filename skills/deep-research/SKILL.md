# Deep Research - 深度研究

## 技能说明

专门处理复杂的、多步骤的研究任务，需要规划、分解和跨工具/文件的长期上下文推理。适合深入分析某个主题。

## 使用场景

1. **大哥要求"调研一下"**：需要深入了解某个主题
2. **复杂问题分析**：涉及多个方面的综合问题
3. **对比分析**：需要对比多个选项
4. **深度报告**：需要产出详细的研究报告

## 执行步骤

### 第一步：理解研究目标

1. 明确研究问题
2. 确定研究深度（浅/中/深）
3. 识别关键维度
4. 设定输出格式

### 第二步：制定研究计划

```
研究计划

目标：XXX

维度：
1. 维度A - 关键问题
2. 维度B - 关键问题
3. 维度C - 关键问题

方法：
- 信息源1
- 信息源2
- ...

输出：报告/表格/建议
```

### 第三步：执行研究

1. 分维度收集信息
2. 交叉验证多个来源
3. 整理关键发现
4. 形成观点

### 第四步：输出报告

结构化输出：
```
## 研究报告：XXX

### 背景
...

### 关键发现
1. ...
2. ...

### 分析
...

### 建议
...
```

## 研究技巧

- 多源验证：一个信息至少两个来源确认
- 数据说话：尽量用数据支撑观点
- 优缺点并重：客观分析
- 可执行建议：最后要有行动建议

## 注意事项

- 研究要深入，不要浅尝辄止
- 保持客观，避免偏见
- 时间控制：不要过度研究
- 明确结论：最后要有清晰结论

## 成功标准

- 研究深入全面
- 信息准确可靠
- 结构清晰易懂
- 建议可执行
