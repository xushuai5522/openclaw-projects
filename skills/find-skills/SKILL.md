# Find-Skills - 技能搜索与安装

## 技能说明

这是 AI 的"元能力"——让 AI 自己学会找技能的能力。装了 find-skills 后，AI 遇到新需求会主动搜索相关技能，而不是用通用知识硬扛。

## 核心理念

| 层次 | 描述 | 特征 |
|------|------|------|
| 第一层 | 工具 | 能力固定，只能做预装的事 |
| 第二层 | 助手 | 能力可扩展，遇到问题会"学习" |
| 第三层 | 伙伴 | 能力自成长，主动学习和更新 |

## 使用场景

1. **大哥布置新任务**：遇到不熟悉的任务类型
2. **现有技能不够用**：预装技能无法满足需求
3. **需要更专业的方案**：想用更专业的技能

## 执行步骤

### 第一步：理解需求

1. 明确任务目标
2. 提取关键词
3. 确定搜索方向

### 第二步：搜索技能

使用 find-skills 搜索相关技能：
```
find "关键词"
```

筛选标准：
- 装机量（越高说明越多人用）
- 评分（>4.5星较好）
- 中文支持
- 最近更新

### 第三步：对比推荐

展示最匹配的 3 个技能：
```
推荐安装 xxx-skills
- 装机量：xxx
- 评分：xxx
- 功能：xxx
是否安装？
```

### 第四步：安装测试

1. 一键安装
2. 配置（如需要 API Key）
3. 测试效果
4. 向大哥汇报结果

## 组合使用

| 场景 | 组合 |
|------|------|
| 日常任务 | find-skills + qmd（快速处理） |
| 复杂任务 | find-skills + task-planning |
| 专业任务 | find-skills + 对应领域技能 |

## 能力增长曲线

| 阶段 | 技能数 | 处理速度 |
|------|--------|----------|
| 开始 | 1个(find-skills) | 15分钟/任务 |
| 1周后 | ~10个 | 12分钟/任务 |
| 2周后 | ~15个 | 8分钟/任务 |
| 1个月后 | 50+个 | 3分钟/任务 |

## 注意事项

- 不是所有技能都需要安装
- 优先装装机量高、评分高的
- 定期清理不常用的技能
- 技能会过时，记得更新

## 成功标准

- 遇到新需求会主动搜索技能
- 能快速找到合适的技能
- 安装后能正常使用
- 能力持续增长
